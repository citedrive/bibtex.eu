Если вы используете BibTeX для создания библиографий для своих научных работ, возможно, вам было интересно, как включить URL-адреса в ваши записи.

Тип поля URL еще не поддерживается официально в BibTeX, хотя он был запланирован в версии 1.0 Ореном Паташником. Тем не менее, дебаты по поводу этой версии и ее намерений восходят к [статье Паташника 2003 года](https://tug.org/TUGboat/tb24-1/patashnik.pdf), которая постепенно заменяется на biblatex с поддержкой Biber.

Таким образом, скорее всего, вам придется использовать одно из следующих решений, например, при цитировании веб-сайтов:

<CiteDrive />

## Используйте тип поля note:

```latex
@misc{...,
  howpublished = "\url{http://...}"
}
```

## Используйте тип поля howpublished:

```latex
@misc{...,
  note = "Доступно по адресу \url{http://...}"
}
```

## Используйте biblatex и его встроенный тип поля URL:

```latex
@online{...,
  url = {http://...}
}
```

Если вы выберете biblatex, вам нужно будет использовать Biber как бэкенд вместо BibTeX. Если вы хотите узнать больше о biblatex и Biber, ознакомьтесь с нашим руководством для начинающих.
