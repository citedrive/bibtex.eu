Поле `series` в BibTeX используется для обозначения серии, к которой принадлежит книга, отчет или другое издание. Указание серии может дать дополнительный контекст для читателя, поскольку оно может быть связано с более широким тематическим или научным дискурсом.

<CiteDrive />

Это поле является необязательным и может использоваться в различных типах записей, включая, но не ограничиваясь, `@book`, `@inbook`, `@incollection` и `@inproceedings`. Оно особенно полезно, когда цитируемая работа является частью продолжающейся серии или более крупного корпуса работ, которые объединяет общая тема, автор или издатель.

При вводе данных в поле `series` рекомендуется использовать полное официальное название серии. Если у серии есть ISSN, также может быть полезно указать его для более точной ссылки.

## Формат

Общий формат включения серии в вашу запись BibTeX следующий:

```bibtex
@entrytype{key,
  ...
  series = {Название серии},
  ...
}
```

## Пример

Вот пример использования поля `series` в записи типа `@book`:

```bibtex
@book{doe2023,
  author = {Джон Доу},
  title = {Передовые исследования в квантовой механике},
  year = {2023},
  publisher = {Scientific Publishing Ltd.},
  series = {Кембриджская серия по физике},
}
```

В этом примере книга является частью "Кембриджской серии по физике", которая может включать различные издания, охватывающие передовые темы в области физики. Указание серии помогает найти работу в рамках более широкого академического обсуждения и может предоставить читателю дополнительные ресурсы по теме.
