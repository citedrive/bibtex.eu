Поле `editor` в BibTeX используется для указания редакторов опубликованного произведения. Это поле важно для таких типов ссылок, как книги, сборники или материалы конференций, где роль редактора отличается от роли автора. Правильное указание редакторов помогает обеспечить корректное атрибутирование и облегчить создание ссылок в академических статьях и работах. Здесь мы более подробно рассматриваем способы указания редакторов в BibTeX.

### Правила форматирования

В поле `editor` принимаются различные форматы записи. Однако форматирование должно быть последовательным и точным, чтобы избежать неправильного представления имени редактора. Вот наиболее часто используемые форматы:

#### Формат: `{Firstname Lastname}`

Это простой способ, когда имя редактора идет перед фамилией. Обратите внимание, что обратный порядок — "Lastname Firstname" — не рекомендуется.

**Пример**

```bibtex
@book{Example2023,
  ...
  editor={Isaac Newton},
  ...
}
```

#### Формат: `{Lastname, Firstname}`

Это рекомендуемый формат, поскольку он четко разделяет имя и фамилию, что особенно полезно для имен с несколькими компонентами.

**Пример**

```bibtex
@book{Example2023,
  ...
  editor={Newton, Isaac},
  ...
}
```

#### Формат: `{Lastname, Suffix, Firstname}`

Для редакторов с суффиксами (например, Jr., Sr., III) имя должно быть записано в формате: фамилия, затем суффикс, затем имя. Все компоненты должны быть разделены запятыми.

**Пример**

```bibtex
@book{Example2023,
  ...
  editor={King, Jr., Martin Luther},
  ...
}
```

### Указание нескольких редакторов

Для указания нескольких редакторов используйте ключевое слово `AND` между именами отдельных редакторов, отформатированных, как указано выше. Это позволяет точно и однозначно указать несколько редакторов.

**Пример**

```bibtex
@book{Example2023,
  ...
  editor={Smith, John AND Doe, Jane},
  ...
}
```

### Рекомендации

1. **Последовательность**: Поддерживайте последовательность в формате записи на протяжении всего файла BibTeX для обеспечения единого представления.
2. **Суффиксы**: Точно указывайте суффиксы, чтобы избежать путаницы, особенно в случаях, когда редакторы имеют семейные отношения.

### Заключение

Поле `editor` является важным элементом в записях BibTeX, особенно для редакторских томов, сборников и материалов конференций. Точное и последовательное форматирование помогает обеспечить правильное атрибутирование редакторов, повышая тем самым достоверность и читаемость ваших ссылок. Помните о соблюдении рекомендуемых форматов для удобного и правильного процесса цитирования.
