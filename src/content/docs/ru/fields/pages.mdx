Поле `pages` в записи BibTeX указывает на конкретные страницы, на которые ссылается работа в рамках более крупного произведения, такого как книга или журнал. Это поле важно при цитировании главы книги или статьи в журнале, поскольку оно помогает направить читателя непосредственно к нужному разделу работы.

## Рекомендации по форматированию

В поле `pages` можно указать одну страницу, диапазон страниц или даже разрозненные страницы. Рекомендуемый способ разделения номеров страниц — с использованием запятой или двойного дефиса. Вот несколько рекомендаций по правильному форматированию поля `pages`:

1. **Одна страница:** Для указания одной страницы достаточно просто написать номер страницы.
    
2. **Диапазон страниц:** Для указания диапазона страниц следует использовать двойной дефис (--). Это обозначает непрерывный диапазон от первой страницы до второй.
    
3. **Несмежные страницы:** Для указания несмежных страниц разные номера страниц или диапазоны следует разделять запятой.
    

## Примеры

Вот несколько примеров использования поля `pages`:

```bibtex
@book{key,
  title = {Название книги},
  author = {Имя автора},
  year = {2023},
  pages = {33--44},   % Диапазон страниц с 33 по 44
}
```


```bibtex
@book{key,
  title = {Название книги},
  author = {Имя автора},
  year = {2023},
  pages = {22, 33--44},   % Страница 22 и диапазон с 33 по 44
}
```

```bibtex
@book{key,
  title = {Название книги},
  author = {Имя автора},
  year = {2023},
  pages = {17},    % Одна страница
}
```

## Примечание

Важно отметить, что поле `pages` не ограничивается только записями типа `@book`, его также можно использовать в других типах записей, таких как `@article` или `@inbook`, чтобы указать соответствующие страницы в этих контекстах. Правильное использование поля `pages` помогает создавать более точные и полезные ссылки.
