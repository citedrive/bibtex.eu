Overleaf — это мощная платформа для LaTeX с предустановленными пакетами, возможностью совместной работы в реальном времени, облачным редактором и более чем 10 миллионами пользователей. Она также интегрируется с такими мощными приложениями, как Writefull, Zotero, Mendeley и [CiteDrive, онлайн-инструментом для управления ссылками на базе BibTeX, который позволяет пользователям работать с одной общей библиографией в Overleaf, синхронизируя все ссылки.](https://www.overleaf.com/blog/better-bibliography-management-with-overleaf-citedrive-and-bibtex-biblatex)

Grammarly — это мощный инструмент, который помогает сделать ваш текст ясным, сжато сформулированным и без ошибок. Теперь использовать Grammarly в Overleaf стало еще проще благодаря последнему обновлению редактора исходного кода до Codemirror 6.

#### Шаг 1: Установите расширение Grammarly для браузера

Первый шаг к использованию Grammarly в Overleaf — установить расширение Grammarly для браузера. Это расширение доступно для Chrome и Firefox и может быть скачано с [сайта Grammarly](https://www.grammarly.com/).

#### Шаг 2: Откройте Overleaf

После установки расширения Grammarly откройте Overleaf и начните новый проект или откройте уже существующий.

#### Шаг 3: Используйте Grammarly в редакторе Overleaf

После установки расширения и открытия Overleaf вы можете использовать Grammarly в редакторе Overleaf. Просто начните вводить текст, и Grammarly автоматически проверит его на грамматические и орфографические ошибки.

#### Шаг 4: Используйте инструмент для проверки текста от Grammarly

Когда вы закончите писать документ, вы можете использовать инструмент для проверки текста Grammarly, чтобы найти оставшиеся ошибки. Просто нажмите на иконку Grammarly в расширении для браузера, затем выберите "Proofread my document" (Проверить мой документ). Grammarly предоставит вам подробный отчет обо всех найденных ошибках с предложениями по их исправлению.

#### Заключение

Использование Grammarly в Overleaf — отличный способ обеспечить безошибочность и качественную обработку текста. С расширением Grammarly для браузера вы можете проверять текст прямо по мере его написания, а затем использовать инструмент для окончательной проверки документа перед его публикацией. Попробуйте, и убедитесь, как это улучшит качество вашего письма.
