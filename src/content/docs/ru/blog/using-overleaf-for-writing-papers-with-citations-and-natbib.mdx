Overleaf — это популярный онлайн-редактор LaTeX, который позволяет пользователям совместно работать над документами в реальном времени. Одной из замечательных особенностей Overleaf является возможность легко добавлять цитаты и библиографии в документы с использованием пакета natbib. В этом посте мы рассмотрим основы использования natbib с Overleaf для добавления цитат и библиографий в ваши работы.

## Добавление цитат

Для того чтобы добавить цитаты в Overleaf, вам нужно сначала добавить пакет natbib в преамбулу документа. Для этого добавьте следующую строку в преамбулу вашего документа:

`\usepackage[numbers]{natbib}`

Это позволит вам использовать команды `\citep` и `\cite` для добавления цитат в ваш документ. Команда `\citep` используется для текстовых цитат и будет отображать цитату как число в скобках, а команда `\cite` используется для внутритекстовых цитат и будет отображать цитату как число.

Например, чтобы сослаться на статью Смит и др. (2020) в тексте, используйте следующую команду:

`Согласно \citep{Smith2020}, это важное открытие.`

Это отобразится как:

> Согласно (Smith et al., 2020), это важное открытие.

## Добавление библиографии

Чтобы добавить библиографию в ваш документ в Overleaf, вам нужно сначала создать файл `.bib`, который будет содержать информацию о всех ссылках, которые вы будете цитировать. Файл `.bib` должен быть сохранен в той же директории, что и ваш основной файл `.tex`. Один из самых простых способов создания этого файла — использование [CiteDrive](https://www.citedrive.com/), инструмента, который подключается к Overleaf и генерирует файл bib для вас. Подробнее об этом можно узнать в [посте на Overleaf.com](https://www.overleaf.com/blog/better-bibliography-management-with-overleaf-citedrive-and-bibtex-biblatex).

После того как вы создали файл `.bib`, вы можете добавить библиографию в ваш документ, добавив следующую команду в ваш документ:

`\bibliography{mybibfile}`

где `mybibfile` — это название вашего файла `.bib`.

Вы также можете указать стиль библиографии, добавив следующую команду в ваш документ:

`\bibliographystyle{plainnat}`

Это отформатирует вашу библиографию в простом стиле. Существует множество различных стилей библиографии, и вы можете найти список всех стилей [здесь](https://www.overleaf.com/learn/latex/Bibliography_styles).
