## Использование расширения для браузера
Преобразовать DOI в цитату BibTeX легко, если вы используете расширение для браузера 'CiteDrive Companion'. Просто выделите DOI мышью, откройте контекстное меню правой кнопкой мыши и выберите 'Search DOI'.

![CiteDrive - Пример](@assets/images/tutorial/browserextension-search-doi.png)

После этого вы увидите предварительный просмотр записи BibTeX, который можете отредактировать перед сохранением в выбранный проект.

![CiteDrive - Пример](@assets/images/tutorial/browserextension-bibtex-preview.png)

Вот и все! Вы можете скачать расширение для следующих браузеров:

* [CiteDrive Companion для Google Chrome](https://chrome.google.com/webstore/detail/citedrive-bibtex-companio/gmmonfphegngpcbcapfbgembkjeookik)
* [CiteDrive Companion для Mozilla Firefox](https://addons.mozilla.org/af/firefox/addon/citedrive-companion/)
* [CiteDrive Companion для Microsoft Edge](https://microsoftedge.microsoft.com/addons/detail/citedrive-bibtex-compan/kpmepfpbjehnlgfbcnbmgihklkkmfnpo)

## Использование онлайн-инструмента
Если вы не используете расширение для браузера, не переживайте — вы все равно можете использовать CiteDrive для преобразования DOI в цитату BibTeX. Просто перейдите на сайт CiteDrive и вставьте свой DOI в поле 'Enter DOI', которое можно найти в разделе 'Search via DOI' при вызове значка плюса в верхней навигационной панели. Затем нажмите на кнопку 'Convert', и вуаля! — ваша цитата BibTeX будет автоматически сгенерирована.

![CiteDrive - Пример](@assets/images/tutorial/citedrive-search-doi.png)
