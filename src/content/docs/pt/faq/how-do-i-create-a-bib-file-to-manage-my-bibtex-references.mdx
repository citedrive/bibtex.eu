## Criando um arquivo .bib manualmente

Criar um arquivo `.BIB` é na verdade bem simples. Tudo o que você precisa fazer é criar um arquivo de texto com a extensão `.bib`. Em seguida, adicione suas referências BibTeX a este arquivo, uma por linha. Por exemplo:

```latex
@article{smith201X,
	title        = {Um artigo interessante},
	author       = {John Smith},
	year         = {201X},
	journal      = {Journal of Interesting Articles}
}
```

Depois de criar seu arquivo `.BIB`, você pode usá-lo em seus documentos LaTeX especificando o comando `\bibliography{nome-do-arquivo}`. Por exemplo:

```latex
\documentclass{article}
\usepackage{natbib}

\begin{document}

...

\bibliography{nome-do-arquivo}

\end{document}
```

É só isso! Criar um arquivo `.BIB` é uma maneira simples de gerenciar suas referências BibTeX. Certifique-se de conferir nossos outros artigos sobre BibTeX para mais dicas e truques!
