[CiteDrive](https://www.citedrive.com/), uma ferramenta para gerenciar e coletar entradas BibTeX e citações, está atualmente desenvolvendo uma extensão para o Obsidian, um aplicativo de anotações com uma abordagem única para a organização de ideias. Ele permite que os usuários conectem rapidamente suas ideias, aumentando a produtividade ao escrever artigos, e reconhecemos a importância dos pesquisadores durante a fase de autoria. O CiteDrive também expandirá o suporte a Markdown, como já faz para Jupyter Notebook, Quarto e R Markdown, para ajudar os pesquisadores a organizar e acessar suas referências em aplicativos como o Obsidian.

# O que exatamente é o Obsidian?
O Obsidian é um aplicativo multiplataforma que permite organizar e conectar rapidamente seus pensamentos e ideias – tudo em markdown. Ele vem com um conjunto completo de ferramentas para criar notas, diagramas, gráficos, documentos e outros tipos de conteúdo. O aplicativo conecta arquivos markdown de maneira inovadora, tornando simples conectar ideias, explorar novos caminhos e expandir rapidamente tópicos.

É uma excelente ferramenta para criar mapas de conhecimento do zero ou a partir de notas já feitas.

O Obsidian foi projetado para ser uma base de conhecimento pessoal, mas também funciona bem em projetos colaborativos. Com o suporte do CiteDrive para o Obsidian, você terá o melhor dos dois mundos: uma ferramenta de pesquisa poderosa e uma base de conhecimento eficiente. Fique atento para mais informações!

# O que é o CiteDrive e para que serve?
O CiteDrive é uma ferramenta de pesquisa baseada em nuvem de última geração que revoluciona o armazenamento e a organização de dados bibliográficos. Este aplicativo permite que você armazene e acesse suas entradas BibTeX de forma segura, de qualquer lugar e a qualquer momento, mantendo-se sincronizado com ferramentas como Overleaf e RStudio. Organize as referências do seu arquivo master.bib global e suas equipes em projetos para facilitar a colaboração com outros pesquisadores, estudantes e professores, mantendo todos na mesma página. Convidamos você a aproveitar os serviços de primeira linha do CiteDrive imediatamente!
