## Usando a extensão do navegador
Converter um DOI para uma citação BibTeX é fácil se você estiver usando a extensão de navegador 'CiteDrive Companion'. Basta selecionar o DOI com o mouse, abrir o menu de contexto com o botão direito do mouse e selecionar 'Pesquisar DOI'.

![CiteDrive - Exemplo](@assets/images/tutorial/browserextension-search-doi.png)

Depois, você verá uma prévia da entrada BibTeX, onde poderá alterá-la antes de salvá-la no projeto de sua escolha.

![CiteDrive - Exemplo](@assets/images/tutorial/browserextension-bibtex-preview.png)

E é isso! Você encontrará o companion disponível para os seguintes navegadores:

* [CiteDrive Companion para Google Chrome](https://chrome.google.com/webstore/detail/citedrive-bibtex-companio/gmmonfphegngpcbcapfbgembkjeookik)
* [CiteDrive Companion para Mozilla Firefox](https://addons.mozilla.org/af/firefox/addon/citedrive-companion/)
* [CiteDrive Companion para Microsoft Edge](https://microsoftedge.microsoft.com/addons/detail/citedrive-bibtex-compan/kpmepfpbjehnlgfbcnbmgihklkkmfnpo)

## Usando a ferramenta online:
Se você não estiver usando a extensão do navegador, não se preocupe - você ainda pode usar o CiteDrive para converter seu DOI para uma citação BibTeX. Basta ir ao site do CiteDrive e colar seu DOI no campo 'Enter DOI', que você encontrará em "Search via Doi" ao clicar no símbolo de mais na barra de navegação superior. Em seguida, clique no botão 'Converter' e voilà! Sua citação BibTeX será gerada automaticamente.

![CiteDrive - Exemplo](@assets/images/tutorial/citedrive-search-doi.png)
