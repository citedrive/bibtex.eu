Il campo `pages` in una voce BibTeX indica le pagine specifiche a cui ci si riferisce all'interno di un'opera più ampia, come un libro o una rivista. Questo campo può essere essenziale quando si cita un capitolo di un libro o un articolo in una rivista, aiutando a indirizzare direttamente i lettori alla sezione pertinente dell'opera.

<CiteDrive />

## Linee Guida per la Formattazione

È possibile specificare una singola pagina, un intervallo di pagine o anche pagine non consecutive. Ecco alcune linee guida per formattare correttamente il campo `pages`:

1. **Pagina Singola:** Se ci si riferisce a una singola pagina, basta scrivere il numero della pagina.
    
2. **Intervallo di Pagine:** Per un intervallo di pagine, è necessario utilizzare i doppi trattini (--) per indicare un intervallo continuo dal primo numero al secondo.
    
3. **Pagine Non Consecutive:** Se ci si riferisce a pagine non consecutive, separare i diversi numeri di pagina o intervalli con una virgola.
    

## Esempi

Ecco alcuni esempi che mostrano come utilizzare il campo `pages`:

```bibtex
@book{key,
  title = {Titolo del Libro},
  author = {Nome dell'Autore},
  year = {2023},
  pages = {33--44},   % Intervallo di pagine dalla 33 alla 44
}
```

```bibtex
@book{key,
  title = {Titolo del Libro},
  author = {Nome dell'Autore},
  year = {2023},
  pages = {22, 33--44},   % Pagina 22 e intervallo dalla 33 alla 44
}
```

```bibtex
@book{key,
  title = {Titolo del Libro},
  author = {Nome dell'Autore},
  year = {2023},
  pages = {17},    % Pagina singola
}
```

## Nota

È importante notare che il campo `pages` non è limitato alle voci `@book`, ma può essere utilizzato anche in altri tipi di voci come `@article` o `@inbook` per indicare le pagine rilevanti in quei contesti. Utilizzare correttamente il campo `pages` aiuta a creare citazioni più accurate e utili.
