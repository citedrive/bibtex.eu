## Creare un file bib a mano

Creare un file `.BIB` è in realtà abbastanza semplice. Tutto ciò che devi fare è creare un file di testo con estensione `.bib`. Puoi quindi aggiungere le tue referenze BibTeX a questo file, una per riga. Ad esempio:

```latex
	@article{smith201X,
		title        = {Un articolo interessante},
		author       = {John Smith},
		year         = {201X},
		journal      = {Journal of Interesting Articles}
	}
```

Una volta creato il tuo file `.BIB`, puoi utilizzarlo nei tuoi documenti LaTeX specificando il comando `\bibliography{filename}`. Ad esempio:

```latex
\documentclass{article}
\usepackage{natbib}

\begin{document}

...

\bibliography{filename}

\end{document}
```

Ecco fatto! Creare un file `.BIB` è un modo semplice per gestire le tue referenze BibTeX. Assicurati di dare un'occhiata ai nostri altri articoli su BibTeX per ulteriori consigli e trucchi!
