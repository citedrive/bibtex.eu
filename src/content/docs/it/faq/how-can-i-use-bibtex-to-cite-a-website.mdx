## Utilizzare BibTeX

In BibTeX usiamo `@misc` come soluzione alternativa e sfruttiamo il campo `note` per specificare l'URL e solitamente una data di accesso. In questo esempio, `url` (*o `hyperref`*) è necessario utilizzando `\usepackage{url}`.

```latex
@misc{CiteDrive2022,
	title        = {CiteDrive porta la gestione delle referenze su Overleaf},
	author       = {CiteDrive, Inc},
	year         = 2022,
	note         = {\url{https://www.citedrive.com/overleaf} [Accesso: (Utilizza la data di accesso)]}
}
```

In alternativa, si può utilizzare il campo `howpublished`:

```latex
@misc{CiteDrive2022,
	title        = {CiteDrive porta la gestione delle referenze su Overleaf},
	author       = {CiteDrive, Inc},
	year         = 2022,
	note         = {Accesso: (Utilizza la data di accesso)},
	howpublished = {\url{https://www.citedrive.com/overleaf}}
}
```

## Utilizzare BibLaTeX

Se utilizzi BibLaTeX, che consigliamo, il tipo di voce `@online` è supportato di default e funziona senza ulteriori modifiche. Puoi inserire l'URL e la data di accesso nei campi `url` e `urldate`, rispettivamente.

```latex
@online{CiteDrive2022,
	title        = {CiteDrive porta la gestione delle referenze su Overleaf},
	author       = {CiteDrive, Inc},
	year         = 2022,
	url          = {https://www.citedrive.com/overleaf},
	urldate      = {Utilizza la data di accesso}
}
```

## Utilizzare `natbib`

Le persone chiedono frequentemente come citare pagine web usando [natbib](/natbib). Sottolineiamo che funziona nello stesso modo di BibTeX. Natbib è uno strumento per la stilizzazione delle citazioni nel testo e dipende da BibTeX come backend.
