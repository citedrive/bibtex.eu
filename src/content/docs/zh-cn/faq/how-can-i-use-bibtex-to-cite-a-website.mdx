## 使用 BibTeX

在 BibTeX 中，我们使用 `@misc` 作为一种变通方法，并利用 `note` 字段来指定 URL，通常还包括访问日期。在这个示例中，`url`（*或 `hyperref`*）是必需的，需使用 `usepackage{url}`。

```latex
@misc{CiteDrive2022,
	title        = {CiteDrive 将参考管理引入 Overleaf},
	author       = {CiteDrive, Inc},
	year         = 2022,
	note         = {\url{https://www.citedrive.com/overleaf} [访问日期: (使用访问日期)]}
}
```

或者，可以使用 `howpublished` 字段：

```latex
@misc{CiteDrive2022,
	title        = {CiteDrive 将参考管理引入 Overleaf},
	author       = {CiteDrive, Inc},
	year         = 2022,
	note         = {访问日期: (使用访问日期)},
	howpublished = {\url{https://www.citedrive.com/overleaf}}
}
```

## 使用 BibLaTeX

如果你使用 BibLaTeX，我们推荐这种方法，`@online` 条目类型天生支持，不需要进一步调整。你可以分别在 `url` 和 `urldate` 字段中输入 URL 和访问日期。

```latex
@online{CiteDrive2022,
	title        = {CiteDrive 将参考管理引入 Overleaf},
	author       = {CiteDrive, Inc},
	year         = 2022,
	url          = {https://www.citedrive.com/overleaf},
	urldate      = {使用访问日期}
}
```

## 使用 `natbib`

人们经常询问如何使用 [natbib](/natbib) 引用网页。我们强调，它的工作方式与 BibTeX 相同。Natbib 是一种用于格式化文内引用的工具，它依赖于 BibTeX 作为后台。
