## 手动创建 Bib 文件

创建 `.BIB 文件` 其实非常简单。你只需要创建一个以 `.bib` 为扩展名的文本文件。然后，你可以将你的 BibTeX 引用逐行添加到该文件中。例如：

```latex
	@article{smith201X,
		title        = {一篇有趣的文章},
		author       = {约翰·史密斯},
		year         = {201X},
		journal      = {有趣文章杂志}
	}
```

一旦你创建了 `.BIB 文件`，就可以在 LaTeX 文档中通过指定 `\bibliography{filename}` 命令来使用它。例如：

```latex
\documentclass{article}
\usepackage{natbib}

\begin{document}

...

\bibliography{filename}

\end{document}
```

就这么简单！创建 `.BIB 文件` 是管理你的 BibTeX 引用的一种简单方法。记得查看我们关于 BibTeX 的其他文章，获取更多技巧和窍门！
