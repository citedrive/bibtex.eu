## 更好的 Bib[La]TeX Overleaf 工作流程

CiteDrive 提供与 Overleaf 的无缝集成，同时支持 BibLaTeX。你可以轻松快速地组织你的参考文献，不必担心字段或类型的限制。无论你使用的是 BibTeX、BibLaTeX、jurabib 还是 JBibTeX，我们都能为你提供支持！

## 新的用户界面和深色模式

根据用户的反馈和我们设计的各种草图和原型，这次更新将大大提升用户体验。记得尝试深色模式！

## RStudio（Posit）中的引用、参考书目和引文管理，在 Quarto 和 R Markdown 中的应用

RStudio 现在与 3.0 版本兼容，这意味着你可以在 Quarto 和 R Markdown 中使用 BibTeX 引用管理。CiteDrive 让你可以将参考文献存储在不同于 Quarto/R Markdown 项目的位置，这样你可以随时轻松更新参考书目，就像在 Overleaf 中一样。
