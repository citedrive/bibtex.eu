# 什么是 Obsidian？
Obsidian 是一款跨平台应用，允许你快速高效地组织和连接你的想法，所有内容都以 Markdown 格式存储。它提供了一整套创建笔记、图表、图形、文档等内容的工具。该应用以新颖的方式连接相关的 Markdown 文件，使得连接想法、探索新路径和快速扩展主题变得简单。

它是从头开始或从已有笔记中创建知识地图的理想工具。

Obsidian 最初是为了个人知识库而设计，但它在协作项目中同样表现良好。借助 CiteDrive 对 Obsidian 的支持，你将获得研究工具和高效知识库的双重优势。敬请关注更多信息！

# 什么是 CiteDrive，它有什么用？
CiteDrive 是一款先进的基于云的研究工具，彻底改变了文献数据的存储和组织方式。这个应用让你可以随时随地安全地存储和访问 BibTeX 条目，并与 Overleaf 和 RStudio 等工具保持同步。将你的全局 master.bib 文件引用和团队组织成项目，以便于与其他研究人员、学生和教师合作，确保每个人都在同一页面上。我们诚挚邀请您立即体验 CiteDrive 的一流服务！
