`volume` 字段在引用较大作品的一部分时至关重要，例如期刊中的文章或多卷书中的章节。这个字段允许你指定引用的是期刊或书籍的哪一卷，从而为读者提供更精确的信息，帮助他们定位被引用的文献。

<CiteDrive />

在许多学术出版物中，期刊通常按卷出版，每卷中包含若干期。卷号通常表示在特定时间段内出版的一系列期刊，通常是按年划分。同样，分为多个卷的书籍或系列也分成不同的卷，每卷包含作品的一部分。

`volume` 字段在引用期刊时尤其重要，因为它使读者能够找到包含被引用文章的特定期刊卷。通常，这个字段与 `number` 字段一起使用，后者进一步指定文章在该卷中的期号。

## 示例

```bibtex
@article{key,
    title={文章标题},
    author={作者姓名},
    journal={期刊名称},
    volume={5},
    number={3},
    pages={123-456},
    year={2022},
}
```

在这个示例中，`volume` 字段指定该文章发表在期刊的第 5 卷中。包含这一信息有助于提供完整且精确的引用，便于读者定位被引用的文献。

类似地，在多卷书籍或系列中，你可以使用 `volume` 字段来指明你引用的章节或部分所在的卷。这同样有助于读者更高效地找到确切的源材料。
