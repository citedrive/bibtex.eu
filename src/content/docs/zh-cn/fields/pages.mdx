在 BibTeX 条目中，`pages` 字段表示在较大作品中引用的具体页码，例如书籍或期刊。当引用书籍的一章或期刊中的一篇文章时，这个字段非常重要，帮助读者直接找到相关的部分。

## 格式化指南

您可以指定单页、页码范围，甚至是非连续的页码。推荐使用逗号或双破折号分隔页码。以下是正确格式化 `pages` 字段的一些指南：

1. **单页：** 如果引用单页，只需写出页码。
    
2. **页码范围：** 对于页码范围，使用双破折号（--）。这表示从第一个数字到第二个数字的连续范围。
    
3. **非连续页码：** 如果引用的是非连续的页码，用逗号分隔不同的页码或页码范围。

## 示例

以下是一些使用 `pages` 字段的示例：

```bibtex
@book{key,
  title = {书名},
  author = {作者名},
  year = {2023},
  pages = {33--44},   % 从33到44页的页码范围
}
```

```bibtex
@book{key,
  title = {书名},
  author = {作者名},
  year = {2023},
  pages = {22, 33--44},   % 引用第22页和33到44页的范围
}
```

```bibtex
@book{key,
  title = {书名},
  author = {作者名},
  year = {2023},
  pages = {17},    % 引用单页
}
```

## 注意

需要注意的是，`pages` 字段不仅限于 `@book` 条目，它也可以用于其他条目类型，如 `@article` 或 `@inbook`，用来标注这些文献中的相关页码。正确使用 `pages` 字段有助于创建更准确、有效的引用。
