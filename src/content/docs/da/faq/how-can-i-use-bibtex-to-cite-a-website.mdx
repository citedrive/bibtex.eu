At citere websites i LaTeX kan gøres på flere måder, afhængigt af hvilken bibliografistil du bruger. I denne guide gennemgår vi de tre mest almindelige metoder: brug af **BibTeX**, **BibLaTeX** og **natbib**. Den nemmeste og mest effektive metode er at bruge **CiteDrive**, som automatisk opretter og håndterer citater i korrekt format.

## 📚 Metode 1: Citer et websted med BibTeX

Hvis du bruger klassisk **BibTeX**, er `@misc` den mest anvendte entry-type til webkilder. Du kan angive URL og adgangsdato i `note`- eller `howpublished`-felterne. For at gøre links klikbare, tilføj `\usepackage{url}` i din preamble.

**Eksempel med `note`:**

```bibtex
@misc{CiteDrive2022,
  title        = {CiteDrive brings reference management to Overleaf},
  author       = {CiteDrive, Inc},
  year         = 2022,
  note         = {\url{https://www.citedrive.com/overleaf} [Adgang: 2025-05-29]}
}
````

**Eksempel med `howpublished`:**

```bibtex
@misc{CiteDrive2022,
  title        = {CiteDrive brings reference management to Overleaf},
  author       = {CiteDrive, Inc},
  year         = 2022,
  howpublished = {\url{https://www.citedrive.com/overleaf}},
  note         = {Adgang: 2025-05-29}
}
```

## ✅ Metode 2: Citer et websted med BibLaTeX (anbefales)

**BibLaTeX** gør det lettere at citere websites med entry-typen `@online`, som har indbyggede felter til `url` og `urldate`. Dette er den mest præcise og rene måde at citere webkilder i LaTeX på.

```bibtex
@online{CiteDrive2022,
  title     = {CiteDrive brings reference management to Overleaf},
  author    = {CiteDrive, Inc},
  year      = 2022,
  url       = {https://www.citedrive.com/overleaf},
  urldate   = {2025-05-29}
}
```

## 🧾 Metode 3: Citering med `natbib` i LaTeX

Hvis du bruger **natbib**, skal du være opmærksom på, at natbib står for selve citationerne i teksten, men stadig bruger BibTeX i baggrunden. Det betyder, at du citerer websites på samme måde som ovenfor – typisk med `@misc`.

I din LaTeX-fil kan du derefter citere med:

```latex
\citet{CiteDrive2022}
```

eller

```latex
\citep{CiteDrive2022}
```

## 🎓 Opsummering: Bedste metode til at citere websites i LaTeX

| Metode    | Entry-type | URL-understøttelse | Anbefalet        |
| --------- | ---------- | ------------------ | ---------------- |
| BibTeX    | `@misc`    | Manuel (`note`)    | ✅ God            |
| BibLaTeX  | `@online`  | Indbygget          | ✅✅ Bedst         |
| natbib    | `@misc`    | Manuel             | ✅                |
| CiteDrive | Alle       | Automatisk         | ✅✅✅ Nem & hurtig |

For akademiske brugere, der ofte citerer websites, er **BibLaTeX + CiteDrive** den mest effektive, nøjagtige og moderne løsning.
