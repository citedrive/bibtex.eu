[CiteDrive](https://www.citedrive.com/)는 BibTeX 항목과 인용을 관리하고 수집하는 도구로, 현재 독특한 노트 작성 방식을 제공하는 노트-taking 앱인 Obsidian을 위한 확장 기능을 개발 중입니다. Obsidian은 사용자가 아이디어를 빠르게 연결하고 연결할 수 있게 하여 논문 작성 시 생산성을 높이는 데 도움을 줍니다. 우리는 연구자들이 저술 단계에서 겪는 중요성을 인식하고 있습니다. CiteDrive는 또한 Obsidian과 같은 앱에서 연구자들이 참고 문헌을 정리하고 접근할 수 있도록 Jupyter Notebook, Quarto, R Markdown에서 이미 지원하고 있는 Markdown 지원을 확장할 예정입니다.

# Obsidian이란 무엇인가요?
Obsidian은 생각과 아이디어를 빠르고 효율적으로 조직하고 연결할 수 있게 해주는 크로스 플랫폼 애플리케이션으로, 모든 작업은 Markdown 형식으로 이루어집니다. 이 앱은 노트, 다이어그램, 그래프, 문서 등 다양한 유형의 콘텐츠를 작성할 수 있는 완전한 도구 세트를 제공합니다. Obsidian은 관련된 Markdown 파일을 독특한 방식으로 연결하여 아이디어를 연결하고, 새로운 경로를 탐색하고, 주제를 빠르게 확장하는 것을 간단하게 만듭니다.

Obsidian은 처음부터 지식 맵을 작성하거나 이전에 작성한 노트를 기반으로 지식 맵을 만들기에 훌륭한 도구입니다.

Obsidian은 개인 지식 기반을 구축하기 위해 설계되었지만, 협업 프로젝트에서도 잘 작동합니다. CiteDrive의 Obsidian 지원으로, 두 가지 세계의 장점을 모두 누릴 수 있습니다: 강력한 연구 도구와 효율적인 지식 기반. 더 많은 정보가 곧 공개될 예정이니 계속 지켜봐 주세요!

# CiteDrive란 무엇이며, 무엇을 위해 사용되나요?
CiteDrive는 bibliographic 데이터를 저장하고 조직하는 방식을 혁신하는 첨단 클라우드 기반 연구 도구입니다. 이 앱을 통해 언제 어디서나 안전하게 BibTeX 항목에 접근하고, Overleaf와 RStudio와 같은 도구들과 동기화된 상태로 작업할 수 있습니다. 글로벌 master.bib 파일의 참고 문헌과 팀을 프로젝트로 조직하여 다른 연구자, 학생, 교사들과의 협업을 쉽게 하고, 모두가 같은 페이지에 있도록 합니다. CiteDrive의 최고급 서비스를 지금 바로 이용해 보세요!
