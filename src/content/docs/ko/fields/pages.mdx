BibTeXの`pages`フィールドは、書籍やジャーナルのような大きな作品内の特定のページを指し示すために使用されます。このフィールドは、書籍の章やジャーナルの記事を引用する際に重要で、読者が関連するセクションに直接アクセスできるようにします。

<CiteDrive />

## フォーマットガイドライン

単一のページ、ページ範囲、または非連続ページを指定することができます。`pages`フィールドを適切にフォーマットするための推奨される方法は以下の通りです：

1. **単一ページ:** 単一のページを指す場合、単にページ番号を書きます。
    
2. **ページ範囲:** ページ範囲を指定する場合は、二重ハイフン（--）を使用します。これは最初の番号から二番目の番号までの連続した範囲を示します。
    
3. **非連続ページ:** 非連続のページを指す場合は、異なるページ番号や範囲をカンマで区切ります。

## 例

以下は、`pages`フィールドの使用例です：

```bibtex
@book{key,
  title = {Title of the Book},
  author = {Author Name},
  year = {2023},
  pages = {33--44},   % ページ範囲 33から44
}
```

```bibtex
@book{key,
  title = {Title of the Book},
  author = {Author Name},
  year = {2023},
  pages = {22, 33--44},   % ページ22と範囲33から44
}
```

```bibtex
@book{key,
  title = {Title of the Book},
  author = {Author Name},
  year = {2023},
  pages = {17},    % 単一ページ
}
```

## 注意

`pages`フィールドは`@book`エントリに限定されるものではなく、`@article`や`@inbook`など他のエントリタイプでも使用できます。これらのコンテキストで関連するページを示すために使用されます。`pages`フィールドを正しく使用することで、より正確で役立つ引用を作成することができます。
