[CiteDrive](https://www.citedrive.com/)は、BibTeXエントリと引用を管理・収集するためのツールであり、現在、ノート作成アプリのObsidian向けの拡張機能を開発中です。このアプリは、ユーザーがアイデアを迅速にリンクして接続できる独自のアプローチを提供し、論文執筆時の生産性を高めます。私たちは、著作段階における研究者の重要性を認識しています。CiteDriveは、すでにJupyter Notebook、Quarto、R Markdown向けに実施しているMarkdownサポートを拡張し、Obsidianのようなアプリで研究者が参考文献を整理し、アクセスしやすくする手助けをします。

# Obsidianとは？
Obsidianは、思考やアイデアを迅速かつ効率的に整理し、接続するためのクロスプラットフォームアプリです。すべてがMarkdownで作成されます。ノート、図、グラフ、文書、その他の種類のコンテンツを作成するための完全なツールセットを備えています。このアプリは、関連するMarkdownファイルを新しい方法で接続し、アイデアをつなげたり、新しい道を探求したり、トピックを迅速に展開したりするのを簡単にします。

これは、ゼロからまたは以前に取ったノートからナレッジマップを作成するのに優れたツールです。

Obsidianは個人的なナレッジベースとして設計されていますが、共同プロジェクトにも適しています。CiteDriveのObsidianサポートにより、強力な研究ツールと効率的なナレッジベースの両方を得ることができます。詳細情報をお待ちください！

# CiteDriveとは何ですか？何に使われますか？
CiteDriveは、文献データの保存と整理を革命的に変える最先端のクラウドベースの研究ツールです。このアプリを使用すると、どこにいても、いつでもBibTeXエントリを安全に保存し、アクセスできます。また、OverleafやRStudioなどのツールと同期を保つことができます。グローバルなmaster.bibファイルの参照とチームをプロジェクトに整理することで、他の研究者、学生、教師とのコラボレーションが容易になり、全員が同じページにいることを保つことができます。ぜひ、CiteDriveの一流サービスをご利用ください！
