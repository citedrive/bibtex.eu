BibTeXの`pages`フィールドは、書籍やジャーナルのような大きな作品の中で参照される特定のページを示すために使用されます。このフィールドは、書籍の章やジャーナルの記事を引用する際に重要で、読者が関連する部分に直接アクセスできるようにします。

<CiteDrive />

## フォーマットガイドライン

`pages`フィールドでは、単一のページ、ページ範囲、または非連続のページを指定できます。推奨されるページ番号の区切り方は、コンマやダブルダッシュを使用することです。`pages`フィールドを適切にフォーマットするためのガイドラインは以下の通りです：

1. **単一ページ:** 単一のページを参照する場合は、単にページ番号を記述します。
    
2. **ページ範囲:** 範囲を指定する場合は、ダブルダッシュ（--）を使用します。これにより、最初の番号から2番目の番号までの連続した範囲を示します。
    
3. **非連続のページ:** 非連続のページを参照する場合は、異なるページ番号や範囲をコンマで区切ります。

## 例

以下は、`pages`フィールドを使用する例です：

```bibtex
@book{key,
  title = {Title of the Book},
  author = {Author Name},
  year = {2023},
  pages = {33--44},   % 33ページから44ページの範囲
}
```

```bibtex
@book{key,
  title = {Title of the Book},
  author = {Author Name},
  year = {2023},
  pages = {22, 33--44},   % 22ページと33ページから44ページの範囲
}
```

```bibtex
@book{key,
  title = {Title of the Book},
  author = {Author Name},
  year = {2023},
  pages = {17},    % 17ページ
}
```

## 注

`pages`フィールドは`@book`エントリに限らず、`@article`や`@inbook`などの他のエントリタイプでも使用して、該当するページを示すことができます。`pages`フィールドを正しく使用することで、より正確で役立つ引用を作成できます。
