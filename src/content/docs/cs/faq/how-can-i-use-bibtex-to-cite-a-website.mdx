Citování webových stránek v LaTeXu lze provést několika způsoby v závislosti na použitém stylu bibliografie. V tomto průvodci pokrýváme tři nejběžnější přístupy: pomocí **BibTeX**, **BibLaTeX** a **natbib**. Nejjednodušší a nejefektivnější způsob doporučujeme přes **CiteDrive**, který automaticky vytváří a spravuje citace ve správném formátu.

## 📚 Metoda 1: Citace webové stránky pomocí BibTeX

Pokud používáte klasický **BibTeX**, pro webové zdroje se běžně používá typ záznamu `@misc`. URL a datum přístupu lze zadat do pole `note` nebo `howpublished`. Pro zobrazení klikacích odkazů přidejte do preambule `\usepackage{url}`.

**Příklad s `note`:**

```bibtex
@misc{CiteDrive2022,
  title        = {CiteDrive brings reference management to Overleaf},
  author       = {CiteDrive, Inc},
  year         = 2022,
  note         = {\url{https://www.citedrive.com/overleaf} [Accessed: 2025-05-29]}
}
````

**Příklad s `howpublished`:**

```bibtex
@misc{CiteDrive2022,
  title        = {CiteDrive brings reference management to Overleaf},
  author       = {CiteDrive, Inc},
  year         = 2022,
  howpublished = {\url{https://www.citedrive.com/overleaf}},
  note         = {Accessed: 2025-05-29}
}
```

## ✅ Metoda 2: Citace webové stránky pomocí BibLaTeX (doporučeno)

**BibLaTeX** zjednodušuje citování webových stránek pomocí typu záznamu `@online`, který má vestavěná pole `url` a `urldate`. Je to nejčistší a nejpřesnější způsob, jak citovat webové stránky v LaTeXu.

```bibtex
@online{CiteDrive2022,
  title     = {CiteDrive brings reference management to Overleaf},
  author    = {CiteDrive, Inc},
  year      = 2022,
  url       = {https://www.citedrive.com/overleaf},
  urldate   = {2025-05-29}
}
```

## 🧾 Metoda 3: Citace webových stránek v LaTeXu pomocí `natbib`

Pokud používáte **natbib**, je důležité vědět, že natbib formátuje citace v textu, ale stále se opírá o BibTeX. To znamená, že webové stránky citujete stejným způsobem jako výše — obvykle pomocí záznamu `@misc`.

Citaci pak do dokumentu vložíte pomocí:

```latex
\citet{CiteDrive2022}
```

nebo

```latex
\citep{CiteDrive2022}
```

| CiteDrive | Všechny     | Automatická       | ✅✅✅ Nejjednodušší |
